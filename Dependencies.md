## Prime field

`secq256r1` (P-256 group order, ~256 bits). Set in
`circomkit.json`. Same prime as
`privacy-ethereum/zkID/wallet-unit-poc/circom/circomkit.json` (the
project this circuit is intended to interoperate with). The `protocol`
field in `circomkit.json` is `"groth16"` only because circomkit
requires a value; **no proofs are generated by this project**.
